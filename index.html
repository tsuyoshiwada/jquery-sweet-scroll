Easy to setup

You can start using it right away with a simple setup

HTML

<!-- Target #header -->
<a href="#header">To #header</a>

<!-- Specify the coordinates -->
<a href="#100">Click to move!</a>

<!-- Using custom data attributes -->
<a href="#" data-target="#header">To #header</a>

JavaScript

// Start by clicking the anchor that begins with "#"
$("a[href^='#']").sweetScroll();

// You can use the some public method.
$.sweetScroll({
  to: 1000
});

Many options

You can be flexibly customized in many settings.

Defaults

$("a[href^='#']").sweetScroll({
  duration: 1200,           //"auto" or specifies animation duration in integer.
  delay: 0,                 //Specifies the milliseconds to start to scroll.
  easing: "easeOutQuint",   //Specifies the easing function of scroll.
  target: null,             //Specifies the target of the scroll position. Example => target: "#header"
  to: null,                 //Specifies the scroll positions.
  offset: 0,                //Adjust the scroll positions.
  autoCoefficient: 2,       //coefficient for "auto" speed.
  stopScroll: true,         //Stop scrolling in any of the events of the wheel or touch move.
  changeHash: false,        //Change the hash after scrolled.
  verticalScroll: true,     //Enable the vertical scroll.
  horizontalScroll: false,  //Enable the horizontal scroll.
  loadedScroll: false,      //Scroll to the position of the hash when the page read.
  loadedScrollQuery: false, //The priority than the "loadedScroll".
  stopPropagation: true,    //Stop the bubbling of anchor click event.
  useVelocity: true,        //if velocity is possible to use from the jQuery or Zepto is used as an alternative to "animate()" method.

  // Callbacks
  beforeScroll: null,
  afterScroll: null,
  cancelScroll: null
});

If you use a custom data attributes

Specifies the option name and its value following the data-.

Result:

Click to move!

Code:

<a href="#" data-easing="easeInOutQuint" data-to="0" data-offset="500">Click to move!</a>

Specified format

various specification format of href and to and offset option.

Code:

<!-- Like the object format => top:[0-9], left:[0-9] -->
<a href="#{top:100,left:0}">Anchor</a>
<a href="#top:100,left:0">Anchor</a>
<a href="#top:500">Anchor</a>
<a href="#" data-to="{top:1000,left:0}">Anchor</a>
<a href="#" data-to="{top:1000}">Anchor</a>
<a href="#" data-to="top:1000,left:200">Anchor</a>
<a href="#" data-to="top:1000">Anchor</a>

<!-- Like the array format => top, left -->
<a href="#500,0">Anchor</a>
<a href="#" data-to="1000,300">Anchor</a>

<!-- Numeric format -->
<a href="#500">Anchor</a>
<a href="#" data-to="500">Anchor</a>

Methods

You can use the some method.

$.fn.sweetScroll("method")

// Disable SweetScroll.
$(selector).sweetScroll("disable");

// Enable SweetScroll.
$(selector).sweetScroll("enable");

// Disable SweetScroll.
$(selector).sweetScroll("destroy");

$.sweetScroll("method")

// Specify the operation at the time of page load.
// You need to run before the DOMContentLoaded event.
$.sweetScroll("loaded", {
  // parametors.
});

// Global Disable SweetScroll.
$.sweetScroll("disable");

// Global Enable SweetScroll.
$.sweetScroll("enable");

// Global Destroy SweetScroll.
$.sweetScroll("destroy");

Zepto

If you want to use from Zepto is the following can be specified for easing.

  • "linear"
  • "ease-in" (the same as the "easeInQuint")
  • "ease-out" (the same as the "easeOutQuint")
  • "ease-in-out" (the same as the "easeInOutQuint")
  • cubic-bezier (...) can not be used.
